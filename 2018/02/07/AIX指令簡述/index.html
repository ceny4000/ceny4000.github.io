AIX指令簡述

一、Grant sudo (給予特定User sudo權限)
輸入 /usr/sbin/visudo
將會開啟編輯sudo file 在底下找到
User privilege specification
root ALL=(ALL) ALL

自己新增user 例如增加
liys ALL=(ALL) ALL

若是希望sudo時不用另外輸入密碼，可以輸入類似
liys ALL=(ALL) NOPASSWD:ALL

若是有特定指令才可允許，可以輸入
liys ALL=(ALL) NOPASSWD:/usr/a.sh, /usr/b.sh

系統會將設定存至 /etc/sudores

二、chown (變更檔案資料夾擁有者)
針對單一檔案/資料夾
chown liys:groupA /tmp/data

包含資料夾內所有檔案/資料夾
chown -R liys:groupA /tmp/data

三、crontab (排程)
要建立crontab建議是預先編輯好一個cron file
用 crontab mycronjobs 指令
/var/spool/cron/crontabs/ 底下會產生一個以當前user命名的檔案，即是crontab內容

※使用時需注意權限及位置，crontab執行指令預設將在/home/username/底下執行

四、nohup (背景執行)
要背景執行一個執行檔可用nohup指令
例如 nohup sh startServer.sh

五、chlang (變更語系)
AIX在預設安裝時語言是en_US，但是在UTF-8環境下會有大量亂碼
要變更語系可以用chlang或是直接編輯/etc/environment
語系文字的大小寫也會影響UTF-8相關設定

繁體中文 (UTF-8) Zh_TW
繁體中文 (UTF-8) ZH_TW
繁體中文 (euc_tw) zh_TW
英文 en_US
英文 (UTF-8) EN_US

六、netstat (網路狀態)
netstat -Aan
查詢目前每個Port的佔用狀況
