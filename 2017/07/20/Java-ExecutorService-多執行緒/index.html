Java ExecutorService 多執行緒

Java 8 開多執行緒

可以用Future確認Thread狀態
例如Future<?> future = executor.submit(//);

利用future.isDone();確認對應Thread是否已完成
利用future.get();來做等待，當此行做完意味著Thread也執行完畢
