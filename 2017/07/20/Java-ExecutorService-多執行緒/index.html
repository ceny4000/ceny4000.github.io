Java ExecutorService 多執行緒

Java 8 開多執行緒

<% codeblock %>
//共開多少thread
ExecutorService executor = null;
try {
executor = Executors.newFixedThreadPool(1);
executor.submit(() -> {
//You’re Source code
});
} catch (Exception e) {

} finally {
try {
//都送出後shutdown，thraed才會回收
executor.shutdown();
} catch (Exception e) {
}
}

<% endcodeblock %>

可以用Future確認Thread狀態
例如Future<?> future = executor.submit(//);

利用future.isDone();確認對應Thread是否已完成
利用future.get();來做等待，當此行做完意味著Thread也執行完畢
