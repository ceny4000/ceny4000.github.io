NSIS使用介紹-手動製作軟體安裝檔

NSIS = Nullsoft Scriptable Install System

NSIS是一款開放程式碼的軟體安裝選單製作程式，安裝時建議不要安裝原版的NSIS，改裝民間版本的NSIS Unicode版本，避免各種不必要的中文編碼問題

NSIS本身要撰寫Script，撰寫完後用NSIS Compile NSI scripts -> Load script，將會產生exe檔案

剛開始撰寫可以抄 C:\Program Files (x86)\NSIS\Examples\ 底下的範例來自行修改

AIX指令簡述

一、Grant sudo (給予特定User sudo權限)
輸入 /usr/sbin/visudo
將會開啟編輯sudo file 在底下找到
User privilege specification
root ALL=(ALL) ALL

自己新增user 例如增加
liys ALL=(ALL) ALL

若是希望sudo時不用另外輸入密碼，可以輸入類似
liys ALL=(ALL) NOPASSWD:ALL

若是有特定指令才可允許，可以輸入
liys ALL=(ALL) NOPASSWD:/usr/a.sh, /usr/b.sh

系統會將設定存至 /etc/sudores

二、chown (變更檔案資料夾擁有者)
針對單一檔案/資料夾
chown liys:groupA /tmp/data

包含資料夾內所有檔案/資料夾
chown -R liys:groupA /tmp/data

三、crontab (排程)
要建立crontab建議是預先編輯好一個cron file
用 crontab mycronjobs 指令
/var/spool/cron/crontabs/ 底下會產生一個以當前user命名的檔案，即是crontab內容

※使用時需注意權限及位置，crontab執行指令預設將在/home/username/底下執行

四、nohup (背景執行)
要背景執行一個執行檔可用nohup指令
例如 nohup sh startServer.sh

五、chlang (變更語系)
AIX在預設安裝時語言是en_US，但是在UTF-8環境下會有大量亂碼
要變更語系可以用chlang或是直接編輯/etc/environment
語系文字的大小寫也會影響UTF-8相關設定

繁體中文 (UTF-8) Zh_TW
繁體中文 (UTF-8) ZH_TW
繁體中文 (euc_tw) zh_TW
英文 en_US
英文 (UTF-8) EN_US

六、netstat (網路狀態)
netstat -Aan
查詢目前每個Port的佔用狀況

DB2指令簡述

一、建立資料庫

AUTOMATIC STORAGE：是否自動增長，預設是YES
PAGESIZE：每個PAGE預設用多少SIZE，應依照目前資料做分析避免空間浪費或效能浪費，預設是4K(4096)

二、建立BufferPool

BUFFERPOOL用途是在資料庫建立一塊儲存在Memory的暫存區，查詢資料會先從此取得增進效能，Database建立時預設會有有個DEFAULT Bufferpool，此處建立多是為了指定給某個TABLESPACE使用，BufferPool主要考量常被存取的資料庫或會在一段時間重複存取固定資料的情況，BufferPool也需要設定SIZE和PAGESIZE，以此例中可以存32768個PAGE

三、TableSpace

TableSpace為劃定一塊區域提供資料作儲存，方便之後控管和備份，150000指的是PAGE數量
若是沒有指定TableSpace，資料都將進入預設的USERSPACE1
每個 TableSpace都可以指定一塊BufferPool做使用，但是同一TableSpace不能指定兩個BufferPool

四、CREATE TABLE 建立資料表

建立資料表的語法，要給予資料表的TableSpace及Inde的TableSpace，PrimaryKey也為Index的一種，若未指定將放進預設的USERSPACE1

五、DB2執行DDL

將檔案丟到db2資料夾的BIN底下直接執行，執行完成後會產生.OUT檔結果
使用時需要注意編碼，在Windows下如有中文，DDL應該為BIG5編碼

Webspere MQ 安全性設定

透過WAS連線到MQ伺服器時，容易有安全性問題，需要設定特定使用者或群組才可使用

流程：
佇列管理程式->通道->點選對應的伺服器連線->MCA->將MCA使用者ID設定為特定User

若是需要變更使用者帳戶群組等，可以Windows->netplwiz 進行設定使用者帳戶

若是對應的User因此有帳戶密碼上的需求，要到Websphere Connection Factory的安全設定，設定特定的User帳號密碼
